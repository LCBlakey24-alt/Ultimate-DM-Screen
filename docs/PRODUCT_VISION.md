# Product Vision

## Working names

- **Rookie Quest Keeper** — current product name.
- **Ultimate DM Screen** — original repository/project name.
- **Rook** — in-app AI/helper identity.

## Core idea

Rookie Quest Keeper is a single campaign companion for tabletop roleplaying groups. It should help a GM prepare, run, and recap sessions while giving players a cleaner way to manage characters and campaign information.

The product should feel like a practical GM screen, a character sheet manager, a campaign notebook, and a lightweight AI assistant all in one place.

## Primary users

### Game Masters

GMs need to:

- Manage campaigns.
- Save lore, NPCs, locations, notes, encounters, maps, events, and handouts.
- Run combat quickly.
- Track party state.
- Generate session prep, recaps, and in-world content from their saved campaign notes.
- Keep rules edition context clear, especially between 2014-style and 2024-style fantasy d20 play.

### Players

Players need to:

- Create characters in different complexity modes.
- Manage character sheets on desktop and mobile.
- Track HP, temp HP, conditions, spell slots, inventory, notes, and progression.
- Join campaigns and eventually receive GM-shared handouts and live-play updates.

## Product principles

1. **Practical first.** Every feature should reduce table friction.
2. **Noisy fantasy UI is bad fantasy UI.** Keep the app readable: dark navy base, gold accents, clear panels, minimal clutter.
3. **Mobile matters.** Player sheets must work properly on phones.
4. **GM trust matters.** AI should reference GM-authored notes and clearly fit the selected campaign/rules context.
5. **Safe rules content.** Avoid protected publisher text. Use SRD-safe mechanics, short labels, original summaries, and user-authored content.
6. **Stable saves beat flashy features.** HP, spells, conditions, notes, inventory, and combat outcomes must persist reliably.

## Long-term feature direction

- Campaign-linked GM and player views.
- Live combat and player sync.
- Player handouts.
- World map pins and travel overlays.
- Better shops, loot, local economies, and settlement tools.
- Homebrew creation workflows.
- Session recap generation from notes and combat logs.
- AI-assisted worldbuilding that uses saved campaign context.
- Exportable/printable campaign material.
